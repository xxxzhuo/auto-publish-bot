# OFFER 格式规则总结

## 已识别的格式类型

### 格式 1: 标准 HK OFFER 格式 (已支持)
```
2026-01-08 HK OFFER 
SAMSUNG DDR4
256*16 K4A4G165WF-BCTD $6.25
```

### 格式 2: 万佳电子格式 (新)
```
万佳电子 
香港现货  实单来砍:
K3KL9L90QM-MFCT          25+  170U
K4F6E3S4HB-KHCL            25+   50U
```
**特点：**
- 开头：公司名 + "香港现货"
- 每行：`料号  DC+  价格 U`
- 价格后缀：`U` = USD
- DC 格式：`25+` 表示 2025 年及以后

### 格式 3: 混合 OFFER 格式 (新)
```
OFFER:
NT6AN256T32AV-J2 usd 19.5,  reel, dc21+ , 1week 
H5CG48MEBDX014N  usd 31.0，dc 25+ 
```
**特点：**
- 开头：`OFFER:`
- 每行：`料号 usd 价格，[描述], dcXX+`
- 包含额外信息：reel, full+par, 1week 等

### 格式 4: 三星 DDR5 RDIMM 格式 (新)
```
2.5 Offer  三星 DDR5 RDIMM
M321R8GA0PB0-CWMCJ    500PCS      $2780  DC25+
M321R4GA3PB0-CWMKJ      1250USD    DC25+
```
**特点：**
- 标题包含：DDR5 RDIMM/UDIMM
- 每行：`料号  数量 PCS  价格  DC`
- 价格格式：`$2780` 或 `1250USD`

### 格式 5: 品牌 + 容量描述格式 (新)
```
三星 LPDDR4 2GB
K4U6E3S4AA-MGCR
23+，80k，37.5 美金，香港

晶存 LPD4 4GB
RS1G32LX4D4BNR-53BT
10k，43U，香港
```
**特点：**
- 标题：`品牌 + 产品类型 + 容量`
- 第一行：料号
- 第二行：`DC+，数量，价格，产地`

### 格式 6: 简单报价格式 (新)
```
H5ANAG8NCJR-XNC      69      25+
H5CG48MEBDX014N     30.0   25+
```
**特点：**
- 每行：`料号  价格  DC+`
- 无货币符号，默认 USD

### 格式 7: SSD 格式 (新)
```
480GB	MZ7L3480HCHQ-00A07	$440.00
960GB	MZ7L3960HCJR-00A07	$660.00
```
**特点：**
- 每行：`容量  料号  价格`
- 容量单位：GB/TB

### 格式 8: 内存条格式 (新)
```
DDR5 64G 5600（500PCS）
三星 M321R8GA0EB0-CWMCH 25+       2450  USD
```
**特点：**
- 标题：`产品类型  容量  速度 (数量)`
- 每行：`品牌  料号  DC+  价格  USD`

### 格式 9: 带产地标注格式 (新)
```
M321R8GA0PB0-CWMXH，产地中国，25+，原封箱，250pcs，2450 美金
M471A2K43EB1-CWE          usd 140.0    dc25+,NON CN
```
**特点：**
- 包含产地信息：`产地中国`, `NON CN`, `CN`, `VN`
- 包装信息：`原封箱`, `full ctn`, `open ctns`

### 格式 10: 数量 + 价格格式 (新)
```
K4B4G1646E-BCNB000   67200  24+/25+  4.9U  
MT41K256M16TW-107:P   100000  25+   4.2U
```
**特点：**
- 每行：`料号  数量  DC+  价格 U`
- 数量较大（几千到几十万）

## 货源地映射规则

| 关键词 | 映射 |
|--------|------|
| `香港`, `HK`, `HKG` | 香港 |
| `大陆`, `CN`, `CHINA`, `中国` | 大陆 |
| `NON CN`, `non cn`, `非中国`, `非 CN` | 大陆 (非中国大陆) |
| `VN`, `越南` | 越南 → 大陆 (海外) |
| `KR`, `韩国` | 韩国 → 香港 (默认) |
| `产地中国`, `产地 CN` | 大陆 |
| `产地 VN`, `产地越南` | 越南 |
| `产地非 CN`, `产地非中国` | 大陆 (非中国大陆) |

## 价格解析规则

| 格式 | 解析 |
|------|------|
| `$6.25`, `$ 6.25` | 6.25 USD |
| `6.25U`, `6.25 USD` | 6.25 USD |
| `6.25 美金`, `6.25 美元` | 6.25 USD |
| `6.25` (无符号) | 6.25 USD (默认) |
| `$BID`, `BID` | 询价 (设为 0.01) |
| `$-`, `-` | 价格待定 (设为 0.01) |

---

## DC (批次) 解析规则

| 格式 | 解析 |
|------|------|
| `25+`, `DC25+`, `dc 25+` | 2025 年及以后 |
| `23+`, `24+/25+` | 对应年份 |
| `full+`, `full` | 全新 |
| `20+` | 2020 年及以后 |

---

## 特殊标记处理

| 标记 | 含义 | 处理 |
|------|------|------|
| `涂标` | 重新打标 | 货物情况=涂标 |
| `reel` | 卷带包装 | 货物情况=Reel |
| `partial` | 散货 | 货物情况=部分 |
| `原封箱`, `full ctn` | 原箱 | 货物情况=全新原箱 |
| `open ctn`, `拆箱` | 开箱 | 货物情况=开箱 |
| `尾数包`, `尾箱` | 尾数 | 货物情况=尾数 |
| `1week`, `2days` | 交期 | 交货天数=7/2 |
| `HK`, `香港` | 货源地 | 货所在地=香港 |
| `CN`, `中国` | 货源地 | 货所在地=大陆 |
| `NON CN` | 非中国 | 货所在地=大陆 (非中国大陆) |

---

## 解析优先级

1. **检测标题行** - 确定整体格式类型
2. **提取品牌分类** - Samsung/SK Hynix/Micron 等
3. **解析产品行** - 根据格式类型选择解析规则
4. **填充默认值** - 企业名称、货主等
5. **货源地映射** - 英文/缩写 → 大陆/香港
