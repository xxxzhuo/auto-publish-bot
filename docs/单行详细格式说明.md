## 🎯 格式特点

### 单行详细格式

**特征：**
- 单行一个产品
- 包含完整信息：料号 + 型号/类型 + 容量 + DC + 价格
- 可能包含品牌中文名
- 价格单位：`u` 代表 USD

**示例：**
```
K4U6E3S4AB-KHCL LPDDR4X 2GB 25+ 48u
K4F6E3S4HM-TFCL LPDDR4 2GB 25+ 48u
RS2G32LO5D4FDB-31BT 晶存 LPDDR5 8GB 25+ 90u
EMMC16G-MW28-GA01 金士顿 16GB eMMC 25+ 22u
H9CCNNNBJTALAR -NVDR 海力士 LPDDR3 2GB 22+ 20u
```

## 🚀 使用方法

### 方法 1: 使用专用解析器

```bash
cd "/Users/mac/Desktop/自动发布机器人"
python3 detailed_parser.py
```

### 方法 2: 使用通用解析器（推荐）

```python
from universal_parser import UniversalOfferParser

text = """
K4U6E3S4AB-KHCL LPDDR4X 2GB 25+ 48u
K4F6E3S4HM-TFCL LPDDR4 2GB 25+ 48u
"""

parser = UniversalOfferParser()
products = parser.parse(text)

print(f"解析 {len(products)} 个产品")
```

### 方法 3: 在 Web 界面使用

1. 打开 `frontend.html`
2. 登录系统
3. 粘贴单行详细格式 OFFER
4. 点击"解析文本"
5. 系统自动识别格式并解析

| **CSV 格式** | `料号，品牌，价格` | universal_parser.py | ✅ |
| **简短格式** | `料号 价格 DC` | universal_parser.py | ✅ |
