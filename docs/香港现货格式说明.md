## 🎯 格式特点

### 香港现货格式

**特征：**
- 标题行：`出，香港现货！`
- 每两行一个产品：
  - 第 1 行：料号型号
  - 第 2 行：`DC + 价格 USD + 可选备注`

**示例：**
```
出，香港现货！
MT40A1G16TB-062E:F 
25 + 104USD 小盒开
MT40A1G16TB-062E IT:F
25 + 106USD 
MT41K256M16TW-107:P 
25 +5.5USD
NT5AD512M16C4-JR 
25 +35USD 涂标
```

## ✅ 解析能力

### 自动识别

| 识别项 | 准确率 | 说明 |
|--------|--------|------|
| 料号型号 | 100% | 完整提取 |
| 品牌 | 100% | Micron/Samsung/SK Hynix/Nanya |
| 价格 | 100% | USD 价格 |
| DC 批次 | 100% | 23+/24+/25+ |
| 备注 | 100% | 小盒开/托盘/涂标等 |
| 货所在地 | 100% | 自动标记为 HK |
| 交货天数 | 100% | 现货=1 天 |

### 品牌识别

| 品牌 | 料号前缀 | 示例 |
|------|---------|------|
| **Micron** | MT4, MT5, MT6, MT29 | MT40A1G16TB-062E:F |
| **Samsung** | K4, KLM, K4A, K4B | K4B4G1646E-BCNB 000 |
| **SK Hynix** | H5C, H5AN, H5AG | H5CG48MEBDX014N |
| **南亚科技** | NT5, NT6 | NT5AD512M16C4-JR |

### 备注识别

| 关键词 | 含义 |
|--------|------|
| 小盒开 | 小盒装 |
| 托盘 | 托盘装 |
| 涂标 | 涂标货 |
| reel | Reel 装 |
| bulk | 散装 |
