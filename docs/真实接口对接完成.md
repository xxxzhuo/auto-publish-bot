## 🔄 错误处理

### 登录失败
```python
# 检查手机号格式
if not mobile or len(mobile) != 11:
    print("❌ 手机号格式错误")

# 检查验证码
if sms_code != "2222":
    print("❌ 验证码错误")
```

### 上传失败
```python
# 检查文件
if not os.path.exists(filepath):
    print("❌ 文件不存在")

# 检查 Token
if not token:
    print("❌ 未登录")
```

### 网络超时
```python
# 自动重试（3 次）
for attempt in range(1, RETRY_COUNT + 1):
    try:
        response = session.post(url, timeout=TIMEOUT)
    except:
        print(f"重试 {attempt}/{RETRY_COUNT}")
```
