# 自动发布机器人 - 配置说明

**重要：** 在运行程序前，请先配置以下参数！

---

## 1. 账号配置

编辑 `config.py`，填写您的账号信息：

```python
# 账号配置
PHONE_NUMBER = "13800138000"  # 您的手机号
PASSWORD = "1234"             # 您的 4 位数密码
```

---

## 2. 接口配置

**需要您提供接口文档信息！**

编辑 `config.py`，填写 API 接口信息：

```python
# 接口配置
BASE_URL = "https://api.example.com"     # API 基础 URL
LOGIN_API = "/api/v1/login"              # 登录接口路径
UPLOAD_API = "/api/v1/upload"            # 上传接口路径
```

### 需要确认的接口信息：

#### Login 接口
- [ ] 接口 URL（完整路径）
- [ ] 请求方法（POST/GET）
- [ ] 请求参数名（手机号字段名、密码字段名）
- [ ] 返回格式（JSON 字段名：token 在哪里？）
- [ ] 是否需要其他参数（验证码、设备信息等）

#### Upload 接口
- [ ] 接口 URL（完整路径）
- [ ] 请求方法（POST）
- [ ] 文件参数名（form-data 中的 name 字段）
- [ ] 是否需要额外的表单数据
- [ ] 是否需要登录态（token/cookie）
- [ ] 返回格式（如何判断成功？）

## 3. 产品文件格式

在 `products/` 目录下放置产品信息的 txt 文件，格式如下：

```txt
产品名称：H5CG48MEBDX014N
品牌：SK Hynix
料号型号：H5CG48MEBDX014N
单价：2.5
货币单位：cny
数量：1000
单位：个
货所在地：深圳
批次（DC）：21+
产品分类：存储芯片
应用：手机
企业名称：鸿达半导体
手机号：13800138000
微信：wechat123
```

**支持的字段：**
- 产品名称、品牌、料号型号、单价、货币单位
- 数量、单位、货所在地、批次（DC）
- 货物情况、产品分类、应用、起订量
- 有效期、交货天数、厂家、重量、尺寸
- 替代品牌、替代型号
- 企业名称、货主、性别、手机号、微信

- **位置：** `output/供需信息_YYYYMMDD_HHMMSS.xlsx`
- **格式：** 与提供的模板一致（27 列标准格式）

- **内容：** 运行日志、错误信息

## 6. 常见问题

### Q: 登录失败
A: 检查 `config.py` 中的手机号、密码、BASE_URL、LOGIN_API 是否正确

### Q: 解析不到产品
A: 检查 `products/` 目录下是否有 txt 文件，格式是否正确

### Q: 上传失败
A: 检查 UPLOAD_API 配置，确认登录态是否有效

### Q: Excel 格式不对
A: 检查 `parser.py` 中的 FIELD_MAPPING，确认字段映射是否正确
