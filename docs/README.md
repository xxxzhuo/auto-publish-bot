# 自动发布机器人

**版本：** v1.0  
**创建日期：** 2026-03-13  
**功能：** 自动登录 → 识别产品信息 → 生成表格 → 上传发布

---

## 🚀 快速开始

### 1. 配置账号和接口

编辑 `config.py`：
```python
PHONE_NUMBER = "13800138000"  # 手机号
PASSWORD = "1234"             # 4 位数密码
BASE_URL = "https://api.example.com"
LOGIN_API = "/api/v1/login"
UPLOAD_API = "/api/v1/upload"
```

### 2. 准备产品文件

在 `products/` 目录下放置产品 txt 文件：
```txt
产品名称：H5CG48MEBDX014N
品牌：SK Hynix
料号型号：H5CG48MEBDX014N
单价：2.5
数量：1000
企业名称：鸿达半导体
手机号：13800138000
```

### 3. 运行程序

```bash
cd "/Users/mac/Desktop/自动发布机器人"
pip3 install -r requirements.txt
python3 main.py
```

## 🔧 模块说明

### login.py - 登录模块
- 自动登录获取 token
- 维护会话状态
- 支持失败重试

### parser.py - 解析模块
- 解析 txt 产品文件
- 支持多种字段名映射
- 自动填充默认值

### excel_generator.py - Excel 生成
- 生成标准格式 Excel
- 自动调整列宽
- 支持追加到现有文件

### uploader.py - 上传模块
- 上传 Excel 到服务器
- 支持附带额外数据
- 失败自动重试

## 🐛 常见问题

**Q: 登录失败？**  
A: 检查 `config.py` 中的手机号、密码、BASE_URL、LOGIN_API

**Q: 解析不到产品？**  
A: 检查 `products/` 目录下是否有 txt 文件

**Q: 上传失败？**  
A: 检查 UPLOAD_API 配置，确认登录态有效
