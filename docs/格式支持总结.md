## 🚀 使用方法

### 方法 1: 自动识别（推荐）
```python
from universal_parser import UniversalOfferParser

parser = UniversalOfferParser()
products = parser.parse(offer_text)

print(f"解析 {len(products)} 个产品")
```

### 方法 2: 专用解析器
```python
# 香港现货格式
from hkstock_parser import HKStockParser
parser = HKStockParser()

# 单行详细格式
from detailed_parser import DetailedLineParser
parser = DetailedLineParser()

# 鸿达格式
from offer_parser import OfferParser
parser = OfferParser()
```

### 方法 3: Web 界面
```bash
# 启动服务
./start.sh

# 打开浏览器
open frontend.html

# 粘贴 OFFER → 解析 → 发布
```
