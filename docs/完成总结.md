## 🔧 启用真实上传

当您需要自动登录和上传时：

### 1. 编辑 config.py

```python
# 账号配置
PHONE_NUMBER = "13800138000"
PASSWORD = "1234"

# 接口配置
BASE_URL = "https://api.example.com"
LOGIN_API = "/api/v1/login"
UPLOAD_API = "/api/v1/upload"

# 运行模式
RUN_MODE = "REAL"  # 从 "SIMULATE" 改为 "REAL"
```

### 2. 提供接口信息

需要确认的接口参数：
- Login 接口：URL、参数名、返回格式
- Upload 接口：URL、文件参数名、额外数据

### 3. 运行

```bash
python3 main.py
```

程序将自动：登录 → 解析 → 生成 → 上传
